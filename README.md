# Designer - Geometric Shape Drawer ­¤ōÉ

This project is a 2D graphics rendering engine developed in **C++**. The program's objective is to process, draw, and export compositions of simple geometric shapes using the concepts of **Polymorphism** and **Virtual Methods**. 

## ­¤ÜĆ Features

- **Multiple Creation Methods**: The 2D drawing can be generated by reading a custom text file containing the shape specifications, or dynamically by calling the inclusion functions directly in the code.
- **SVG Export**: Any drawing processed by the class can be exported to an **SVG** (Scalable Vector Graphics) file, allowing high-quality visualization without resolution loss in any modern web browser.
- **Subpixel Coordinate System**: It uses a Cartesian system of real numbers (double), with the origin `(0.0, 0.0)` at the top-left corner and the Y-axis increasing downwards, ensuring high precision in positions and dimensions.
- **Supported Shapes**: The system supports rendering 6 fundamental geometric shapes with independent outline and fill properties:
  - Lines (`<line>`)
  - Circles (`<circle>`)
  - Rectangles (`<rect>`)
  - Triangles (`<polygon>`)
  - Rhombuses (`<polygon>`)
  - 5-point Stars (`<polygon>`)
- **Restricted Polymorphic Architecture**: The program reads parameters, creates, and processes the saving of shapes without using generic conditionals (`if/switch`) when exporting the data, delegating the behavior exclusively to the virtual methods of each object.

## ­¤øĀ’ĖÅ Structure and Main Classes

- **`Shape`**: Abstract base class for all shapes. It stores the color (RGB), the outline width, and the virtual methods `read` and `save`.
- **`FilledShape`**: Abstract class that inherits from `Shape`. It adds support for shapes that can contain an internal fill color (all except the line).
- **Concrete Classes** (`Line`, `Circle`, `Star`, `Rectangle`, `Rhombus`, `Triangle`): Each inherits from its respective base class, having its own constructors and overriding the `read` (to fetch specific vertices, centers, and radii) and `save` (to translate the shape's math into standard SVG XML tags) functions.
- **`Designer`**: The main manager class of the application. It contains the proportions of the drawing dimensions and a dynamic vector of pointers to the shapes (`std::vector<Shape*>`), centrally managing the batch reading and export of the drawing.
